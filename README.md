🛒 Ecommerce Django Project

A full-stack Ecommerce Web Application built using Django.
This project includes product listing, cart system, user authentication, and order management.

🚀 Live Project
👉 (add later when deployed)

🧑‍💻 Tech Stack
Python 🐍
Django 🌐
SQLite 🗄️
HTML / CSS / Bootstrap 🎨
JavaScript ⚡

✨ Features
🔐 User Registration & Login
🛍️ Product Listing Page
🛒 Add to Cart / Remove from Cart
📦 Order Placement System
💳 Payment Page UI (Razorpay ready integration)
🧾 Order Success Page
🖼️ Product Images & Categories
👤 User-based Cart System

📁 Project Structure
miniecommerce/
│
├── myapp/              # Main app
├── miniecommerce/      # Project settings
├── media/              # Product images
├── templates/          # HTML files
├── static/             # CSS/JS/images
├── db.sqlite3
└── manage.py

⚙️ Installation & Setup
# 1. Clone repo
git clone https://github.com/vjawahar9-cloud/ecommerce-project.git

# 2. Move into project
cd ecommerce-project

# 3. Create virtual environment
python -m venv venv

# 4. Activate environment
venv\Scripts\activate   # Windows

# 5. Install dependencies
pip install -r requirements.txt

# 6. Run server
python manage.py runserver
